# Snake Evolution
Snake Evolution is a 2D arcade snake game made with Python and Pygame.
This project is a modern version of the classic Snake game. The player controls a snake that moves around the screen and eats food to grow bigger. The goal is to survive as long as possible and get the highest score.
Unlike the classic Snake game, this version adds new gameplay features. The game includes upgrades and power ups that give the snake special abilities. Some power ups make the snake faster, some protect it with a shield, and others increase the score.
The game also uses procedural levels. This means the level layout can change each time you play. Because of this, every run feels a little different.
The game also includes a settings menu, an achievements system, and a high score system that saves your best score.

## Requirements
Python 3
Pygame library

## Installation
Install Python on your computer.
Open Command Prompt.
Type the following command to install Pygame
pip install pygame
Download the project files.
Run the game with the following command
python snake_evolution.py

## Controls
Arrow keys move the snake
Escape opens the pause menu

## Project Information
Language used in this project is Python.
Graphics and game logic are built using the Pygame library.
The entire game is written in a single Python file.
